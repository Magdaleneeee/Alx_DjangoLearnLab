## Posts & Comments API

### Posts
- GET /api/posts/
- POST /api/posts/
- PUT /api/posts/{id}/
- DELETE /api/posts/{id}/

### Comments
- GET /api/comments/
- POST /api/comments/
- PUT /api/comments/{id}/
- DELETE /api/comments/{id}/

### Permissions
- Only authors can edit or delete their posts and comments
- Authentication is required for all endpoints
